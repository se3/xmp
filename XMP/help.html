This part of the Xtreamer Mod Pack, a small summary about the usage.

In the left menu in the order of the appearance:

MAVVY FS
This PHP script is the base of the entire Web page. (Mavvy old fakeshellje)

SKINS
Select the skin what you want then click on the install under in the middle
Be patient, maybe needs some seconds to install.

FONTS

You can change the base GUI font. Just put your font into /xmp/guiffont library, after open the font webpage in the XMP. IMPORTANT: always check the uploaded fonts CRC, becaue it can crash the system on the Xtreamer.
